## Technologies
- NodeJS
- Node Package Manager (NPM)
- Hapi module
- Nanoid module
- Nodemon module
- Eslint module

## Install
- Install NodeJS.
```shell
sudo apt install nodejs
```
- Install Node Package Manager (NPM).
```shell
sudo apt install npm
```
- Install NPM modules.
```shell
npm install
```

## Run
```
npm run dev
```
