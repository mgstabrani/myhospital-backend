## Technologies
- NodeJS
- Node Package Manager (NPM)
- Express
- Nanoid module
- Nodemon module
- Eslint module

## Install
- Install NodeJS.
```shell
sudo apt install nodejs
```
- Install Node Package Manager (NPM).
```shell
sudo apt install npm
```
- Install NPM modules.
```shell
npm install
```
- Create .env file. The file must contain these variables.
```
DB_NAME=myhospital
DB_USERNAME=myhospital
DB_PASSWORD=mypassword
DB_HOST=localhost
DB_PORT=3306
```

## Run
```
npm run dev
```
